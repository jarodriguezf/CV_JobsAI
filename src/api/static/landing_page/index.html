Acerca del proyecto

Este proyecto trata de automatizar a traves de la IA, el proceso de criba de curriculums. Usamos NLP para extraer la informacion mas relevante en base a los datos aportados.

  • Podremos insertar curriculums y ofertas laborales.
  • En funcion de una oferta laboral, se mostrara los cv mas adecuados a dicha oferta.
  • Podremos indicar el grado de similitud en funcion de la relevancia semantica en cada oferta laboral
  • El proyecto ha sido optimizado para cv y ofertas de las areas tecnologicas.
  • El proyecto se ha realizado con fines didacticos (no comercial).
